# Flappy Bird Game 🎮

A simple Flappy Bird-style game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Classic Flappy Bird gameplay
* Two themes:

  * 🌱 Classic Mode
  * 🌌 Space Mode
* Play button with countdown
* Score tracking
* Keyboard controls (Space / Arrow Up)

## 🎮 How to Play

* Press **Space** or **Arrow Up** to jump
* Avoid hitting the pipes
* Try to get the highest score!

## 🛠️ Tech Stack

* HTML5 Canvas
* CSS
* JavaScript
